# RZFrame Web - 极简图片边框工具 📸
__给你的照片多一点仪式感。__
## 📖 简介 (Introduction)
**RZFrame Web** 是一个基于浏览器的轻量级图片边框处理工具，专为摄影爱好者设计。

目的是区别于市面上众多复杂的编辑器，它具有以下独特优势：

* **无需下载 APP**
- **无需注册登录**
* **纯前端运行**

  用户的照片不会上传到服务器，最大程度保护隐私。且针对移动端进行了优化，尽量实现如原生应用般的丝滑体验。

## ✨ 核心特性 (Features)

**📱 移动端优先体验**
* 专为触屏优化的交互
- 沉浸式全屏预览模式
* 防误触与触感反馈设计

**📷 智能 EXIF 读取**
* 自动解析参数（相机/手机品牌、型号、镜头、光圈、快门、ISO）PS：如果照片有信息的话，没有则不显示
- 支持手动修正与自定义（提供一行文字编辑，默认显示解析的照片信息，用户也可以删除自行编辑任何其他内容）

**🎨 边框模版**
* **Classic:** 经典画廊白框/黑框
- **Cinema:** 经典画廊白框/黑框
- **Float:** 独特的悬浮投影+底图毛玻璃效果
- **固定比例选择:** 预设了几种固定比例模板，以满足如微信朋友圈，小红书，ins等社交媒体的常见图片比例

**🔠 文本/Logo 排版**
* Logo可以上传任何图案，且支持颜色反转
- 支持两种自定义字体样式（无衬线/有衬线）
* 内容支持切换显示/不显示
* 支持**左/中/右**独立对齐

**⚡ 性能与清晰度**
* 支持 2K 分辨率 高清导出，适配朋友圈、小红书、Ins等社交媒体手机浏览
- 自动压缩导入照片的分辨率，防止导入过高分辨率图片导致的卡顿等问题
